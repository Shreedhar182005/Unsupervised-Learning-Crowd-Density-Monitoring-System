# Crowd Density Monitoring System

## Overview
Crowd Density Monitoring System is a computer vision-based application that detects and monitors crowd density in real-time using video feeds. The system helps identify overcrowded areas and can be used in public places, events, transportation hubs, and smart city applications.

## Features
- Real-time crowd monitoring
- Human detection using computer vision
- Crowd density estimation
- Density status classification (Low, Medium, High)
- Live video processing
- User-friendly interface
- Automated alerts for overcrowded areas

- Machine Learning / Deep Learning
